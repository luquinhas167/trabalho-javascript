#  Calculadora de Planos de Saúde

Aplicação web interativa desenvolvida em **HTML, CSS e JavaScript** para comparar preços de planos de saúde de duas operadoras (A e B), considerando:

* Idade do usuário
* Peso e altura (para cálculo do IMC)
* Fator de comorbidade
* Tabelas de preço das operadoras
* Custo-benefício
* Gráfico comparativo (Chart.js)

A aplicação exibe automaticamente o **plano mais vantajoso** com base nos valores calculados.

---

##  Tecnologias Utilizadas

* **HTML5**
* **CSS3** (responsivo e moderno)
* **JavaScript** (lógica completa do cálculo)
* **Chart.js** (para gerar o gráfico comparativo)
